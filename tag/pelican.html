  1. Python相关：
    教程里涉及到诸多命令。这些命令需要将相关路径都放到系统变量里，才可以直接运行。 比如，在cmd控制台中，直接输入pelican-quickstart，如果提示不能识别，这个时候就要检查pelican的安装路径是不是在系统变量Path下，如果没有，赶紧加上。

    Tips：修改完系统变量，需要重启cmd控制台。

  2. 启动本地调试
    生成好内容之后，进入output目录，利用Python自带的SimpleHTTPServer运行

    cd output
    python -m SimpleHTTPServer # 注意大小写 `
